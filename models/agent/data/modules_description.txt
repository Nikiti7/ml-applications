1) Модуль text
   - функционал: анализ текста, sentiment-analysis, TF-IDF, ключевые слова
   - технологии: HF transformers, sklearn
   - вход: строка; выход: метка тональности, вероятности

2) Модуль image
   - классификация изображений ResNet18 (pretrained)
   - pipeline: resize → normalize → модель → softmax
   - технологии: PyTorch, torchvision
   - выход: класс и вероятность

3) Модуль audio
   - преобразование аудио → текст (STT)
   - whisper или TensorFlow STT
   - выход: транскрипт + базовый NLP

4) Модуль video
   - ffmpeg / OpenCV: извлечение кадров
   - каждый кадр пропускается через image-pipeline
   - поиск ключевых сцен (порог по изменению кадров)

5) Модуль llm (GPT-2)
   - генерация текста
   - настройка: max_length, top_p, top_k, do_sample

6) Модуль api
   - FastAPI эндпоинты:
     /analyze_text
     /predict_image
     /predict_audio
     /predict_video
     /generate_text
     /rag_query
   - возвращает JSON

7) RAG агент
   - TF-IDF retrieval
   - ruT5 base multitask генерация
   - детерминированные fallback-ответы

8) Web-UI
   - HTML + CSS
   - FastAPI + Jinja2
   - чат с агентом
