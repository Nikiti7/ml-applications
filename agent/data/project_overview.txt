Проект "ML Applications" создан для демонстрации интеграции различных моделей машинного обучения и локальных языковых моделей (LLM).

Основная цель проекта — разработать интеллектуального ИИ-агента, который объединяет модули анализа текста, аудио, изображений и видео в единую систему.

Структура проекта:
- text — обработка текста и анализ тональности;
- audio — распознавание речи (Speech-to-Text);
- image — классификация изображений;
- video — анализ содержимого видеороликов;
- api — REST API на FastAPI для демонстрации моделей;
- agent — интеллектуальный агент с поиском и генерацией ответов;
- docs — отчёт и документация.

Результат: агент способен отвечать на вопросы по проекту, использовать локальные документы, запускаться через веб-интерфейс и работать полностью офлайн.
