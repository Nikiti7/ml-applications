Модуль text — анализирует текст с помощью модели BERT, проводит классификацию по тональности.
Модуль audio — использует TensorFlow и библиотеку SpeechRecognition для преобразования аудио в текст.
Модуль image — применяет сверточную нейронную сеть (ResNet18) для классификации изображений.
Модуль video — извлекает ключевые кадры из видео и классифицирует содержимое при помощи модели из image-модуля.

Каждый модуль имеет свой main.py, набор тестовых данных и README.md с описанием.
