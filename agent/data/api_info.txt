API-сервисы разработаны на FastAPI.

1. Sentiment Analysis API:
   - POST /analyze — анализ одного текста.
   - POST /batch-analyze — пакетный анализ.
   Использует pipeline("sentiment-analysis").

2. LLM API:
   - POST /generate — генерация текста по prompt.
   Модель: cointegrated/rut5-base-multitask или gpt2.

Swagger UI доступен по адресу /docs.
