Для тестирования используется pytest и FastAPI TestClient.
Настроен GitHub Actions workflow:
- при каждом commit автоматически запускаются тесты API;
- проверяется успешный ответ и корректность формата JSON.

Тесты охватывают:
- Sentiment API;
- LLM API (генерация текста);
- обработку ошибок и некорректных запросов.
