ИИ-агент — центральный элемент проекта. Он объединяет все модули через RAG (Retrieval-Augmented Generation):
1. Извлекает нужные документы из базы (ChromaDB).
2. Формирует ответ с помощью LLM (cointegrated/rut5-base-multitask).
3. Может работать через CLI и веб-интерфейс (FastAPI).

Используемые технологии:
- LangChain-концепция (RAG);
- ChromaDB — хранение векторных представлений документов;
- Transformers — генерация ответов;
- FastAPI — веб-интерфейс;
- HTML/JS — визуализация в браузере.

Агент понимает вопросы о проекте, структуре, коде и принципах работы моделей.
